# Git

操作Git，脑海里有操作不同指针的画面，需要理解下面几类指针

- index文件：包含当前追踪的文件hash

- 文件指针（blob）：文件hash
- 目录指针（tree）：目录结构hash
- commit指针（commit）：版本hash
- branch指针（master、dev……）：指向某个版本hash
- HEAD指针：指向某个版本hash
