## 解决hexo的MarkDown渲染器与MathJax的冲突
编辑 node_modules\marked\lib\marked.js
```js
escape: /^\\([\\`*{}\[\]()# +\-.!_>])/,
// 替换为
escape: /^\\([`*\[\]()# +\-.!_>])/,
// 取消对\\,\{,\}的转义


em: /^\b_((?:[^_]|__)+?)_\b|^\*((?:\*\*|[\s\S])+?)\*(?!\*)/,
// 替换为
em:/^\*((?:\*\*|[\s\S])+?)\*(?!\*)/,
// 取消对对斜体标记 _ 的转义
```
